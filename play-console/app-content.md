# 앱 콘텐츠 양식 답변 (Play Console → 정책 → 앱 콘텐츠)

각 항목 좌측이 Play Console 질문, 우측이 입력값/답변이다. 출처가 명확하지 않은 항목은 _(결정 필요)_로 표기.

## 1. 개인정보처리방침

| 항목 | 값 |
|---|---|
| URL | `https://jeiel85.github.io/pulput-ink-android/privacy-policy.html` |

> 위 URL은 이 저장소의 `docs/privacy-policy.html`을 GitHub Pages로 게시한 결과다. 게시 절차는 [github-pages-setup.md](github-pages-setup.md) 참고.

## 2. 앱 액세스 권한

| 질문 | 답변 |
|---|---|
| 앱의 일부 기능에 로그인이나 기타 액세스 권한이 필요합니까? | **아니요** — 모든 기능을 별도 로그인 없이 사용. AI 기능은 사용자가 자신의 API 키를 앱 설정에 입력하는 방식이며, Google 검수자가 별도 테스트 계정 없이 모든 기능을 검토할 수 있음. |

> 만약 검수 중 Google 측에서 API 키가 필요하다고 거절하면, 검수용 임시 키를 발급해 "테스트 계정" 칸에 메모로 제공한다.

## 13. 인앱 결제

현재 인앱 결제 없음. AAB 매니페스트에 `BILLING` 권한 없음을 확인 (`app/src/main/AndroidManifest.xml`). 향후 프리미엄 모델 다운로드 등을 유료화할 경우 별도 등록 필요.

## 14. 권한 사용 사유 (Play Console "민감한 권한" 자동 검사 대비)

| 권한 | 사용 사유 |
|---|---|
| `INTERNET` | OpenAI Whisper / Google Gemini API 호출, Whisper 모델 파일 다운로드 |
| `RECORD_AUDIO` | 사용자가 직접 트리거한 설교 녹음. 백그라운드 녹음 없음. 녹음된 음성은 사용자 단말 내에만 저장되며, AI 전사 요청 시에만 사용자가 명시적으로 동의한 외부 API로 전송됨. |

> Play Console은 별도의 `Sensitive Permissions` 양식을 요구하지 않음 (현재 권한이 background location / SMS / call-log 등이 아니기 때문).
