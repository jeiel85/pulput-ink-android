
## 전체 설명 (최대 4000자)

```
Pulpit Ink는 설교자, 신학생, 그리고 말씀을 글로 남기고 싶은 모든 분을 위한 모바일 설교 비서입니다. 마이크 한 번으로 설교 음성을 녹음하면, OpenAI Whisper와 Google Gemini AI가 결합되어 음성 → 필기 노트 변환 → 성경 구절 정렬 → 설교 개요(아웃라인)까지 한 흐름으로 정리해 줍니다.

【주요 기능】

■ 한 번의 탭으로 시작하는 설교 녹음
강단 위, 사무실, 차 안 어디서든 마이크 권한만 허용하면 즉시 고음질 녹음이 시작됩니다. 녹음 중 실시간 상태 표시로 마이크 입력 여부를 한눈에 확인할 수 있습니다.

■ AI 자동 필기 및 받아적기 (OpenAI Whisper)
녹음이 끝나면 Whisper 모델이 한국어/영어 설교 음성을 정확하게 받아적어 필기 노트로 옮겨 줍니다. 클라우드 API 모드와 단말 내 로컬 모델(tiny / base / small) 모두 지원하므로, 인터넷이 없는 환경에서도 동작합니다.

■ 성경 구절 자동 정리 (Gemini AI)
설교 본문 안에 등장한 성경 인용을 자동으로 찾아내고, 「창세기 1:1」, 「요한복음 3:16」과 같은 표준 표기로 다듬어 줍니다. 굵게 처리, 한글 성경 책 이름 표준화, 띄어쓰기 교정까지 한 번에 처리합니다.

■ 설교 아웃라인 자동 생성
필기된 본문을 바탕으로 Gemini가 설교 개요(서론·본론·결론, 핵심 메시지, 적용점)를 마크다운 형식으로 정리해 줍니다. 다시 컴파일 버튼으로 언제든 재생성 가능합니다.

■ 설교 보관함 & 검색
모든 설교는 단말 안의 데이터베이스에 저장되며, 제목·본문·태그로 빠르게 검색할 수 있습니다. 외부 서버에 사용자의 설교를 영구 저장하지 않습니다.

■ 편집·복사·공유
필기된 문장을 블록 단위로 직접 수정할 수 있고, 본문/아웃라인/전체 패키지를 클립보드로 복사하거나 다른 앱(메모, 노션, 카카오톡 등)으로 공유할 수 있습니다.

【이런 분께 추천합니다】

- 매주 설교를 준비하는 목사·전도사·간사
- 설교를 글로 남겨 공동체와 나누고 싶은 평신도 사역자
- 강의·세미나·성경공부 필기가 필요한 신학생
- 음성 메모를 구조화된 문서로 자동 변환하고 싶은 모든 분

【개인정보·보안】

녹음된 음성과 필기된 텍스트는 기본적으로 단말 안에만 저장됩니다. AI 필기·정리 기능을 사용할 때만 해당 데이터가 OpenAI 및 Google Gemini의 API로 전송되어 처리되며, 전송은 모두 HTTPS로 암호화됩니다. 자세한 내용은 개인정보처리방침을 확인해 주세요.

【API 키 안내】

본 앱은 OpenAI Whisper API와 Google Gemini API를 사용합니다. 클라우드 필기 / Gemini 기능을 사용하려면 본인의 API 키 등록이 필요합니다. 로컬 Whisper 모델만 사용하실 경우 API 키 없이도 오프라인 설교 필기가 가능합니다.
```
