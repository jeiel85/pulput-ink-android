# IARC 콘텐츠 등급 설문 답변

Play Console → 정책 → 앱 콘텐츠 → 콘텐츠 등급 → 설문 시작

## 카테고리 선택

**유틸리티, 생산성, 커뮤니케이션 또는 기타** (Reference, news, or educational 도 고려)

→ 권장: **참고자료, 뉴스 또는 교육** (Reference, News, or Educational)

## 설문 답변 (모두 "아니요")

| 카테고리 | 질문 | 답변 |
|---|---|---|
| 폭력 | 폭력적인 행위가 포함되어 있습니까? | 아니요 |
| 폭력 (만화/판타지) | 만화/판타지 폭력 | 아니요 |
| 성적 콘텐츠 | 성적인 콘텐츠 / 노출 | 아니요 |
| 욕설 | 욕설 / 비속어 | 아니요 |
| 약물 | 마약·담배·알코올 관련 | 아니요 |
| 도박 | 도박 / 시뮬레이션 도박 | 아니요 |
| 공포 | 공포 / 무서운 콘텐츠 | 아니요 |
| 사용자 상호작용 | 사용자 간 실시간 상호작용 / 채팅 | 아니요 |
| 사용자 위치 공유 | 사용자의 위치를 다른 사용자에게 공유 | 아니요 |
| 사용자 개인정보 공유 | 사용자 개인정보(이름·연락처)를 다른 사용자에게 공유 | 아니요 |
| 디지털 구매 | 디지털 콘텐츠/기능을 구매할 수 있는가 | 아니요 |
| 인터넷 액세스 | 제한 없는 인터넷 액세스 (브라우저 등) | 아니요 |

## 예상 결과

- **IARC**: 3+ / Everyone
- **한국 (GRAC)**: 전체 이용가
- **북미 (ESRB)**: Everyone
- **유럽 (PEGI)**: PEGI 3
- **호주 (ACB)**: G

## 비고

본 앱은 사용자 본인의 음성 녹음과 AI 기반 텍스트 처리만 수행하며, 사용자 간 콘텐츠 공유 / 채팅 / 위치 / 결제 기능이 없으므로 가장 낮은 등급으로 분류된다.
