# 데이터 보안 양식 (Play Console → 정책 → 앱 콘텐츠 → 데이터 보안)

## 데이터 수집 및 공유 요약

| 질문 | 답변 |
|---|---|
| 앱이 필수 사용자 데이터를 수집하거나 공유합니까? | **예** |
| 수집된 모든 데이터가 전송 중에 암호화됩니까? | **예** (HTTPS / TLS) |
| 사용자가 데이터 삭제를 요청할 수 있습니까? | **예** — 앱 내 설교 보관함에서 항목 삭제, 단말에서 앱 제거 시 로컬 데이터 전체 삭제 |

## 데이터 유형별 상세

### A. 오디오 파일

| 항목 | 값 |
|---|---|
| 수집 여부 | **예** (단말 내 저장) |
| 공유 여부 | **예** — 사용자가 "전사" 버튼을 누른 경우에 한해 OpenAI 또는 Google Gemini API로 전송 |
| 수집 목적 | 앱 기능 (전사) |
| 필수/선택 | **선택사항** — 녹음은 사용자가 직접 트리거 |
| 이유 | 설교 음성을 텍스트로 변환하기 위함 |

### B. 음성 녹음에서 파생된 텍스트

| 항목 | 값 |
|---|---|
| 수집 여부 | **예** (단말 내 저장) |
| 공유 여부 | **예** — 사용자가 "AI 교정", "아웃라인 재생성" 등을 누른 경우 Google Gemini API로 전송 |
| 수집 목적 | 앱 기능 (성경 구절 정리, 아웃라인 생성) |
| 필수/선택 | **선택사항** |

### C. 사용자가 직접 입력한 텍스트 (설교 제목, 주제 등)

| 항목 | 값 |
|---|---|
| 수집 여부 | **예** (단말 내 저장만) |
| 공유 여부 | **아니요** (외부 전송 없음) |
| 수집 목적 | 앱 기능 (보관·검색) |
| 필수/선택 | **선택사항** |

### D. 다음 항목은 **수집하지 않음**

- 이름, 이메일, 전화번호 등 개인 식별 정보
- 정확한 위치 / 대략적 위치
- 결제 정보
- 사진 / 동영상 (오디오만 사용)
- 연락처, 캘린더, SMS, 통화 기록
- 앱 활동 분석 / 광고 식별자
- 기기 ID (Advertising ID, IMEI 등)
- 진단 데이터 / 크래시 로그 (별도 분석 SDK 미사용)

## 데이터 처리 관행

| 항목 | 답변 |
|---|---|
| 모든 데이터가 전송 중에 암호화되는가? | **예** (HTTPS) |
| 사용자가 데이터 삭제를 요청할 수 있는가? | **예** (앱 내 삭제 + 앱 제거) |
| Play Families 정책 준수 | 해당 없음 (만 18세 이상 대상) |
| 독립적인 보안 검토를 받았는가? | 아니요 |

## 제3자 데이터 처리자

| 처리자 | 처리 데이터 | 목적 | 정책 링크 |
|---|---|---|---|
| OpenAI | 오디오 파일 | Whisper 전사 (선택 기능, 사용자 API 키 사용) | https://openai.com/policies/privacy-policy |
| Google (Gemini API) | 텍스트 | 성경 구절 교정·아웃라인 생성 (선택 기능, 사용자 API 키 사용) | https://ai.google.dev/terms |

> 사용자가 자신의 API 키를 직접 등록해 사용하므로, 각 처리자의 데이터 보존·삭제 정책은 해당 사용자의 OpenAI/Google 계정 설정에 따른다.
