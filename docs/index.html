Features

강단에서 문서까지, 한 흐름.

탭 한 번으로 시작되는 녹음부터 마크다운 아웃라인까지 — 설교자가 직접 하던 반복 작업을 AI가 대신합니다.

🎙

One-tap 녹음

강단·차·서재 어디서든 마이크 한 번이면 시작. 실시간 상태 표시로 입력 여부를 한눈에 확인합니다.

✦

AI 자동 전사

OpenAI Whisper로 한국어·영어 설교를 정확히 옮깁니다. 인터넷 없이도 동작하는 단말 내 모델(tiny/base/small) 지원.

✝

성경 구절 정리

Gemini AI가 본문 속 인용을 찾아 「창세기 1:1」 같은 표준 표기로 다듬고 볼드 처리합니다.

≡

아웃라인 자동 생성

서론·본론·결론·핵심 메시지·적용점까지 마크다운으로 정리. 언제든 재컴파일.

⌕

보관함 + 검색

모든 설교를 단말 내부에 저장. 제목·본문·태그로 즉시 검색.

⇪

편집·복사·공유

블록 단위 편집, 클립보드 복사, 다른 앱(노션·카톡·메모)으로 공유.

How it works

음성에서 잉크로, 네 단계.

목회자가 자기 손으로 하던 일을 따라가는 흐름. 각 단계에서 멈추거나, 편집하거나, 다시 컴파일할 수 있습니다.

1

Record

설교를 녹음합니다. 백그라운드 녹음은 없으며 사용자가 직접 시작·종료합니다.

2

Transcribe

Whisper가 오디오를 텍스트로 옮깁니다. 클라우드 또는 로컬 모델 선택 가능.

3

Refine

Gemini가 성경 구절을 표준 표기로 다듬고 띄어쓰기를 교정합니다.

4

Outline

전사 본문을 바탕으로 설교 아웃라인을 마크다운으로 자동 생성합니다.

Privacy

설교는 사적인 것이라야 합니다.

Pulpit Ink는 분석 SDK·광고 SDK·계정 시스템을 일체 포함하지 않습니다. 모든 데이터는 기본적으로 단말 안에서만 흐릅니다.

No Cloud
By Default
  • 녹음·전사·아웃라인 모두 단말 내 Room DB에만 저장
  • 회원가입·로그인·계정 시스템 없음
  • 광고 식별자(AAID)·위치·기기 ID 일체 수집 없음
  • 분석/크래시 SDK 미탑재
  • AI 기능 호출 시에만 사용자가 등록한 API 키로 OpenAI / Google에 HTTPS 전송
  • 로컬 Whisper 모델 사용 시 전 과정 오프라인 가능
