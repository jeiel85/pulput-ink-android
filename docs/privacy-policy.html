Pulpit Ink 개인정보처리방침

최종 업데이트: 2026-05-24

본 정책은 모바일 애플리케이션 Pulpit Ink (이하 "본 앱", 패키지명 io.pulpit.ink)이 사용자의 개인정보를 어떻게 처리하는지 설명합니다.

1. 운영자 정보

  • 앱 이름: Pulpit Ink
  • 개발사: AI Studio
  • 문의 이메일: pedaiah85@gmail.com

2. 수집하는 정보

본 앱은 사용자가 직접 생성하는 다음 데이터를 사용자 기기 내부에서만 처리합니다.

  • 사용자가 녹음한 설교 오디오 파일 (.m4a, .wav 등)
  • 음성에서 변환된 텍스트 (전사문)
  • 사용자가 입력한 설교 제목, 주제 메모, 태그
  • 사용자가 입력한 OpenAI / Google Gemini API 키 (앱 설정 영역에 저장)

본 앱은 다음 정보를 수집하지 않습니다.

  • 이름, 이메일, 전화번호 등의 개인 식별 정보
  • 위치 정보 (정확/대략 모두 미사용)
  • 연락처, 캘린더, 사진 라이브러리, SMS, 통화 기록
  • 광고 식별자 (AAID), IMEI 등 기기 고유 식별자
  • 사용 로그, 크래시 분석, 행동 분석 데이터 (별도 분석 SDK 미탑재)

3. 데이터 저장 위치

수집한 모든 데이터는 사용자 단말 내의 Android Room 데이터베이스와 앱 전용 저장소(internal storage)에 저장되며, 본 앱 개발사의 서버로 전송·저장되지 않습니다. 사용자가 앱을 삭제하면 모든 로컬 데이터가 함께 삭제됩니다.

4. 제3자 처리 (선택 기능)

사용자가 명시적으로 "AI 전사", "AI 교정", "아웃라인 생성" 등의 기능을 호출하는 경우에만, 해당 기능에 필요한 데이터(오디오 또는 텍스트)가 다음 외부 서비스의 API로 HTTPS를 통해 전송됩니다.

  • OpenAI — Whisper 음성 인식 API. 처리 데이터: 오디오 파일. 정책: openai.com/policies/privacy-policy
  • Google (Generative AI / Gemini) — 텍스트 교정 및 요약. 처리 데이터: 전사된 텍스트. 정책: ai.google.dev/terms 및 policies.google.com/privacy

위 API 호출은 사용자가 본인 명의로 발급한 API 키를 사용해 사용자 본인의 OpenAI / Google 계정 책임 하에 수행됩니다. 본 앱이 제공하는 로컬 Whisper 모델(tiny/base/small)을 사용하면 위 외부 전송 없이 단말 내에서만 전사가 수행됩니다.

5. 권한 사용

  • android.permission.INTERNET — OpenAI / Gemini API 호출 및 Whisper 모델 파일 다운로드
  • android.permission.RECORD_AUDIO — 사용자가 명시적으로 시작한 설교 녹음. 백그라운드 녹음 없음.

6. 데이터 보안

외부 API 호출은 모두 HTTPS(TLS)로 암호화됩니다. 로컬 데이터는 Android 운영체제의 표준 앱 샌드박스로 보호되며, 단말 자체 암호화 설정에 따릅니다.

7. 만 14세 미만 아동의 개인정보

본 앱은 만 18세 이상 성인 사용자(목회자, 신학생, 사역자)를 대상으로 합니다. 의도적으로 만 14세 미만 아동의 개인정보를 수집하지 않습니다.

Pulpit Ink — Privacy Policy (English)

Last updated: 2026-05-24

This policy describes how the mobile application Pulpit Ink (package io.pulpit.ink) handles user information.

1. Operator

  • App name: Pulpit Ink
  • Developer: AI Studio
  • Contact: pedaiah85@gmail.com

2. Information collected

The app processes the following user-generated data only on the user's device:

  • Sermon audio recordings (.m4a, .wav, etc.)
  • Transcripts derived from the audio
  • Titles, topic notes, and tags entered by the user
  • OpenAI / Google Gemini API keys entered by the user

The app does not collect:

  • Personal identifiers such as name, email, or phone number
  • Location (neither precise nor approximate)
  • Contacts, calendar, photo library, SMS, or call log
  • Advertising identifiers or device identifiers
  • Usage analytics or crash data (no analytics SDK is bundled)

3. Storage

All collected data is stored in the app's local Android Room database and internal storage. None of it is transmitted to or stored on servers operated by the developer. Uninstalling the app removes all local data.

4. Third-party processing (optional features)

Only when the user explicitly invokes "AI transcription", "AI proofreading", or "outline generation", the relevant data (audio or text) is sent over HTTPS to:

  • OpenAI — Whisper speech-to-text API. Data: audio file. Policy: openai.com/policies/privacy-policy
  • Google (Generative AI / Gemini) — text formatting and summarization. Data: transcribed text. Policy: ai.google.dev/terms and policies.google.com/privacy

These calls use the user's own API keys, billed to and governed by the user's own OpenAI / Google accounts. If only the bundled local Whisper models (tiny / base / small) are used, transcription happens entirely on-device with no external transmission.

5. Permissions

  • android.permission.INTERNET — calls to OpenAI / Gemini APIs and downloads of Whisper model files
  • android.permission.RECORD_AUDIO — sermon recording, started only by explicit user action. No background recording.

6. Security

All external API calls use HTTPS (TLS). Local data is protected by the standard Android app sandbox and benefits from device-level encryption when enabled.

7. Children

The app targets adult users (pastors, seminary students, ministers) aged 18+. It does not knowingly collect data from children under 14.

8. User rights

  • Delete individual sermons from the in-app library
  • Clear API keys (blank them in app settings)
  • Uninstall the app to remove all local data

Inquiries: pedaiah85@gmail.com

9. Changes

Any change to this policy will be published on this page with an updated "Last updated" date. Material changes will be announced in the app update at the time of release.
